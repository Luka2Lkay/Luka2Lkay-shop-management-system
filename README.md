# Shop Management System

The system manages employees and managers within a shop. This frontend utilizes Angular for user interaction.

Key features include adding/removing managers and employees, assigning employees to managers, and a user-friendly interface with confirmation popups for deletions. The system prioritizes security and scalability while ensuring a smooth user experience.

# How to run this app

## Prerequisites

- Nodejs and npm (or yarn)
- Angular Cli

## Installation

- Clone the project
- Navigate into the project's directory and run **npm install** in the terminal for the installation of all the dependencies
- Make sure the server is running. Click this [link](https://github.com/Luka2Lkay/Luka2Lkay-shop-management-system-api/tree/main) and go through this project's README.md file for instructions.
- Run **ng serve -o** in the terminal to execute the application
